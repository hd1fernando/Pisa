# Pisa
A chrome extension that generates Italian utilities

- Fiscal Code
- IVA
- Phone Number

![sample](assets/img/sample.png)

# How to use
1. [Download](https://github.com/hd1fernando/Pisa/tags) de latest tag.
2. Unzip.
3. Enable develop mode on Chrome.
4. Load the unzipped folder.
